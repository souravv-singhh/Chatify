### Description
**Chatify** : A chat application using LRU(Least Recently Used Cache). It provides CRUD Operations in constant time.

### Implemented:
  - Caching concept.
  - LRU(Least Recently Used) Cache.
  - Linked List & defualt HashMaps.

### Tech. Stack:
HTML, CSS, JavaScript
